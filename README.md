## B) Project Title
 HanLearn : Smart Mandarin Learning App

## C) Introduction

HanLearn is a hybrid mobile learning application that helps beginners learn Mandarin in an intuitive and engaging manner. It includes features like translation tools, vocabulary practice, quizzes, and progress monitoring, making language learning more accessible and enjoyable for everyone. The app includes stimulating tasks and quizzes to focus on fundamental language skills such as expansion of vocabulary, translation, and practice.

### **Problem**: 
Learning a new language can be challenging, especially without the proper resources or tools. Mandarin, with its complex characters and tones, is especially difficult for beginners. This app aims to simplify Mandarin learning by offering an easy-to-use platform with essential language-learning features.

### **Motivation**: 
With globalization, learning Mandarin has become an essential skill for many. HanLearn provides an easy way for users to begin their Mandarin journey, without the need for expensive courses or materials.

## D) Objectives of the Proposed Mobile App

1. Provide beginners with a simple but engaging environment for learning Mandarin.
2. Allow users to practice translating between English and Mandarin, in order to strengthen language acquisition.
3. Make a classified vocabulary library of vital words from daily speech, education, travel, and food.
4. Provide entertaining quizzes and practice modules to assist users reinforce their knowledge while they keep track of their learning progress.
5. Track learning progress with an easily understood visual representation of completed quizzes, scores, and vocabulary learnt.

## E) Target Users

1. Beginners interested in learning Mandarin:
   This app is designed for people who are interested in learning mandarin language. All ranges of ages can use this app due to its simple and interactive modules to help users begin their journey consistently. 
2. Language learners looking for an affordable tool:
   HanLearn provides a cost-effective alternative to traditional Mandarin courses, focusing on essential vocabulary and conversational skills.

## F) Features and Functionalities

### 1. User Authentication (Firebase)
Login/Register: Users can sign in or create an account using email and password.
User Profile: Personalized profiles include the user's name and learning level.
Use Firebase Authentication for managing user login and registration

### 2. Translation Module
English and Mandarin Translation: Instant translations between English and Mandarin.
Translation Details: Displays Chinese characters, Pinyin, and English meanings.
Vocabulary Saving: Users can save their translated words into the "My Vocabulary" section for later reference.
Use Firestore database usage for saving translations

### 3. Vocabulary Bank
Categorized Vocabulary: Words are organized into categories like Daily Conversation, Education, Travel, and Food.
Detailed Vocabulary: Each word includes the character, Pinyin, and its meaning.
Apply a well-structured data model with ListView UI for displaying words

### 4. Quiz & Practice Module
Multiple Quiz Types: Includes multiple choice, match words to meaning, fill in the blank, and guess the correct translation.
Features: Displays immediate feedback after each question, tracks scores, and stores quiz results.
State management for effective quiz handling and result tracking

### 5. Learning Progress Tracker
Track Learning: Keep track of completed quizzes, scores, and vocabulary learned.
Progress Visualization: A simple progress bar or percentage display to show learning progress.
Use Firestore integration for data tracking with visual progress representation
